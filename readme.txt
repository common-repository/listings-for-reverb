Embed Reverb.com listings and search bar on Wordpress

== Description ==

Embed Reverb.com listings and search bar on Wordpress. This is available through two short codes [reverb-listings] or [reverb-search].

You can embed all the listings from your Reverb.com shop, or select from specific listing types of search queries.

We support all the listing properties avaliable here:
https://reverb.com/page/embedded-listings


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/reverb` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add the short code tag [reverb-listings] or [reverb-search] to your word press posts.
